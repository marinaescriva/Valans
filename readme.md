## Project overview

The objective of this project was to design and implement a landing page presenting an **urban electric mobility product**.

The structure of the page is divided into four main sections:

1. Hero section  
2. Product information panels  
3. Product presentation section  
4. Contact form  
5. Footer

The design focuses on:

- clear product communication
- modern mobility aesthetics
- responsive layout
- simple interactions

---

## Responsive design

Two breakpoints were designed and implemented:

### Desktop
- Width: **1600px**
- Grid: **12 columns**
- Margin: **100px**
- Gutter: **16px**

### Mobile
- Width: **402px**
- Grid: **4 columns**
- Margin: **20px**
- Vertical scroll layout
